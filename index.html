Applied Visual Design in CSS: Geometry

Scale CSS Function

The scale function is capable of resizing an element relative to its original size with a multiplying factor. Scale will change the width and height so they are proportionate to each other.It is used with the transform property, like this:

.div1 {transform: scale(1.5);}

This will scale up the size of the div element with the class div1 by 1.5. The following example will demonstrate this using images.

The left image has no scaling at all. The right image has scaling of 1.5. However, this function can have two parameters, allowing you to scale the height and width by different values. It would be simply written like this:

.div2 {transform: scale(1.25, 1.5)}

This function can also be used to scale down elements in the same way.

Hover Selector

The :hover selector allows you to style elements when your mouse hovers over them. You may notice it if you hover your mouse over any of the buttons in the navigation bar. For example, if you wanted the text size of all paragraph elements to increase when a cursor hovers over it, you would write it like this:

p:hover {transform: scale(1.25);}

You can see how it works in the preview below:

The hover selector can be used with any css property, but keep in mind that those properties only become applied when you hover over the element.

SkewX and SkewY Functions

The skewX() and skewY() functions will skew an element along their respective axis by the given angle. It is another one of the many 2D transform functions that can be used with the transform property. The syntax is the same as any other CSS function, but you want to be using the deg unit to specify the angle you want to skew the element by. So, if you wanted to skew one element by 20 degrees on the X-axis and another element by 20 degrees on the Y-axis, you would write it like this:

#div1{transform: skewX(20deg)} #div2{transform: skewY(20deg)}

Graphics in CSS

In CSS, it is possible to create many different graphics if you have good knowledge of CSS properties. For example, with borders, you can create a triangle that looks like this:

This can be achieved by changing the height and width and manipulating the borders to create a triangle shape, like this:

            #triangle {
            width: 0;
            height: 0;
            border-style: solid;
            border-top: 50px solid transparent;
            border-bottom: 50px solid transparent;
            border-left: 87px solid #FF3700;
            border-right: 0;
          }

The shape here is technically a rectangle. How this works is that this tiny square has a huge border and zero width or height, leaving behind just the border. Then you want to use different border properties (border-top, border-bottom, etc.) to extract a triangle out of the border. This can be done with different dimensions to make any kind of triangle.
